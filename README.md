# :construction_worker: How work ?
```bash
# Clone this repository
$ https://github.com/KaikySantos/wildbeast.git

# Access the project folder
$ cd wildbeast

# Install the dependencies
$ yarn install

# Run the application
$ yarn start
```
Go to http://localhost:3000/ to see the result.
